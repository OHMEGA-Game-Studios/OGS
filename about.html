About Omega Game Studios

Omega Game Studios is a small game development studio that was founded in 2021. Formerly known as MegaBox Game Studios, we have been developing games for over 7 years. Still working on our first public game, The Ruined Manifest. You will enjoy playing it as much as we have enjoyed making it.

About Our Game

The Ruined Manifest is a survival horror game, this game is set in the Cascade Mountains of Washington State. We are calling it the Thrashaw Hills. This game will tell a story about a Natural event that has caused the forest to grow so massive. This event is nammed The Ruined Manifest. This game will be available for PC, MAC on first release. Though later in development Linux will be added to our support list.

  • Where will I find The Ruined Manifest 0.1.0?

    • Answer: Version 0.1.0 and other Pre-Releases will work on Mac and Windows but version 1.0.0 will be available on Mac, Windows, and Linux.

  • Will there be a launcher?

    • Answer: Yes! Once The Ruined Manifest is out of Pre-Release we will have an official launcher for the game. Additionally we will add any other games we create to the launcher!

  • When will The Ruined Manifest be released?

    • Answer: The Ruined Manifest 1.0.0 will be released somtime in 2025

  • Who will be able to access version 0.1.0?

    • Answer: Anyone! You can download The Ruined Manifest for FREE on our website during the Pre-Release stage.

  • Will version 1.0.0 be free?

  • Will The Ruined Manifest work on my laptop?

However most MACS should be able to run it.
